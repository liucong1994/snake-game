# 贪吃蛇游戏

一个使用HTML5 Canvas和JavaScript实现的经典贪吃蛇游戏。

## 游戏特点

- 现代化的UI设计
- 响应式布局
- 键盘控制
- 分数系统
- 游戏结束提示

## 如何开始

1. 点击"开始游戏"按钮
2. 使用键盘方向键（↑↓←→）控制蛇的移动
3. 吃到食物可以增加分数和蛇的长度
4. 避免撞到墙壁或自身

## 技术栈

- HTML5
- JavaScript
- Tailwind CSS

## 本地运行

直接在浏览器中打开`index.html`文件即可开始游戏。
